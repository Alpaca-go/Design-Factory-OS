## 0. GPT Execution Core

**原始 Logo：** Locked，不得修改、重绘、拆解或替换。
**Creative Authority：** Maximum。除 Logo、客观事实和项目显式锁定项外，其余视觉资产均可重新设计。

### 当前核心问题

用 2～4 句话说明原方案最影响品牌质量的问题，不展开重复分析。

### 唯一视觉升级命题

“从 ______ 升级为 ______。”

用一句话补充：保留什么，通过什么具体设计机制升级什么，同时避免什么。

### 核心视觉锚点

- 仅列 3～5 个真正具有品牌记忆价值的锚点

### 新视觉关键词

- 使用能够对应具体视觉动作的关键词，避免空泛形容词

### Anchor Image

说明第一张图片必须建立的色彩、材质、光线、行业属性、构图和品牌气质。

### 图片生成顺序

1. Anchor Image
2. Packaging System
3. Brand Context
4. Campaign Poster
5. Detail / Craft

### 关键禁止事项

- 修改、重绘、拆解、替换或伪造 Logo
- 编造品牌、行业、产品或合规事实
- 平均继承全部旧资产
- 回到常见行业模板
- 每张图片重新选择一种风格
- 用无业务意义的 Mockup 代替真实触点
